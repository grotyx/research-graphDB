## 채점 기준 (Rubric)

### 1. 정확성 (Clinical Correctness)

> "답변에 의학적으로 **틀린 내용**이 있는가?"

| 점수 | 기준 | 예시 |
|:---:|------|------|
| **5** | 모든 주장이 정확함. 오류 없음 | 올바른 적응증, 정확한 합병증률, 정확한 수술 기법 설명 |
| **4** | 거의 정확. 사소한 부정확 1건 (임상 결정에 영향 없음) | 연도가 틀렸으나 결과는 정확 |
| **3** | 대체로 정확하나 부정확 2-3건 | 합병증률이 다소 부정확하지만 방향성은 맞음 |
| **2** | 중요한 부정확 있음. 임상 결정에 영향 가능 | A가 B보다 우수하다고 했으나 실제는 유사함 |
| **1** | 심각한 오류 있음. 임상적으로 위험한 결론 | 적응증을 반대로 기술, 합병증을 효과로 기술 |

### 2. 완전성 (Completeness)

> "이 질문에 답하기 위해 **중요한 포인트를 빠뜨리지 않았는가?**"

| 점수 | 기준 | 예시 |
|:---:|------|------|
| **5** | 모든 중요 포인트 포함 (outcomes, complications, evidence level) | 비교 질문에서 양쪽 모두의 결과 + 합병증 + 근거 수준 포함 |
| **4** | 대부분 포함. 부차적 포인트 1-2개 누락 | 주요 outcome은 있으나 cost-effectiveness 누락 |
| **3** | 핵심은 포함하나 중요 포인트 일부 누락 | TLIF vs PLIF에서 fusion rate는 있으나 complication 누락 |
| **2** | 중요한 측면 상당 부분 누락 | 비교 질문인데 한쪽 수술만 기술 |
| **1** | 매우 불완전. 핵심 포인트 대부분 누락 | 질문과 무관한 내용 위주 |

### 3. 근거 품질 (Evidence Quality)

> "인용된 논문의 **근거 수준이 높은가?** RCT/MA 위주인가, Case Report 위주인가?"

| 점수 | 기준 | 예시 |
|:---:|------|------|
| **5** | MA/SR/RCT 위주 (Level 1a-1b). 최신 고품질 근거 | 3편 이상 RCT/MA 인용 |
| **4** | 고품질 근거 포함하되 Cohort도 혼재 | RCT 1-2편 + Cohort 다수 |
| **3** | Cohort/Case-control 위주 (Level 2a-2b) | RCT 없이 retrospective 위주 |
| **2** | Case series/report 위주 (Level 3-4) | 높은 근거 수준 논문 없음 |
| **1** | 근거 없음 또는 EL 불명 | 인용 없이 일반적 서술, 또는 출처 확인 불가 |

### 4. 인용 신뢰성 (Citation Trustworthiness)

> "인용된 논문이 **실제로 존재**하고, 인용된 주장을 **실제로 뒷받침하는가?**"

| 점수 | 기준 | 예시 |
|:---:|------|------|
| **5** | 모든 인용이 존재하며 주장을 정확히 뒷받침 | paper_id로 확인 가능한 실제 논문, 내용 일치 |
| **4** | 대부분 확인 가능. 1건 정도 불확실 | 9/10 논문 확인됨, 1편 매칭 불확실 |
| **3** | 일부 인용 불확실하거나 주장과 논문 내용 불일치 | 논문은 있으나 해당 결과를 보고하지 않은 경우 |
| **2** | 다수 인용 불확실. 존재하지 않는 논문 포함 가능 | Author-year로만 인용, DB에서 확인 안 됨 |
| **1** | 인용 대부분 확인 불가. Hallucination 의심 | 존재하지 않는 저자/연도 조합 다수 |

### 5. 임상 유용성 (Clinical Usefulness)

> "이 답변을 바탕으로 **실제 임상 결정**을 내릴 수 있는가?"

| 점수 | 기준 | 예시 |
|:---:|------|------|
| **5** | 바로 임상 결정에 활용 가능. 근거 수준 명시, 구체적 수치 포함 | "Level 1a 근거에 의하면 UBE의 ODI 개선이 3.2점 우수 (p=0.02)" |
| **4** | 대체로 유용. 약간의 추가 검색 필요 | 방향성은 명확하나 구체적 수치 일부 부족 |
| **3** | 참고 가능하나 추가 문헌 검토 필요 | 여러 관점 제시되나 결론이 불명확 |
| **2** | 제한적 유용성. 추가 검토 없이 결정 불가 | 너무 일반적이거나 구체성 부족 |
| **1** | 유용하지 않음. 임상 결정에 도움 안 됨 | 부정확하거나 관련 없는 내용 |
