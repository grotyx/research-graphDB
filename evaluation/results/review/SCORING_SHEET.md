# Baseline 비교 채점표

> 각 질문별로 4개 baseline 답변을 읽고 채점해주세요.
> 각 항목 0~5점 (0=매우 나쁨, 5=매우 우수)

## 채점 기준

| 항목 | 설명 |
|------|------|
| **정확성 (Correctness)** | 답변 내용이 의학적으로 정확한가? 잘못된 주장이 있는가? |
| **완전성 (Completeness)** | 중요한 근거를 빠뜨리지 않았는가? 양쪽 비교가 균형 있는가? |
| **근거 품질 (Evidence Quality)** | 인용된 논문의 근거 수준이 높은가? (RCT/Meta > Cohort > Case) |
| **인용 신뢰성 (Citation Trust)** | 인용된 논문이 실제로 존재하고 주장을 뒷받침하는가? |
| **임상 유용성 (Usefulness)** | 실제 임상 결정에 도움이 되는가? |
