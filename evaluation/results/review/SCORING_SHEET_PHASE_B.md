# Phase B: Expert Validation Scoring Sheet

> **목적**: 4개 검색 시스템의 답변 품질을 전문의가 blinded 평가
> **평가자**: ________________ (전공: ________, 경력: ____년)
> **평가일**: ________________
> **Blinding**: 각 답변은 System A/B/C/D로 표시 (실제 시스템은 평가 완료 후 공개)
>
> **평가 파일**: `blinded/questions/DG-XXX_blinded.md` 참조
> **점수 기록**: `blinded/scoring_blinded_(이름).csv` 에 기록

---

## 평가 방법

1. `blinded/questions/` 폴더에서 답변 파일을 열고 **System A~D 답변을 순서대로** 읽음
2. 아래 **Rubric(R1~R5)**에 따라 각 항목별 점수 부여 (각 1-5점)
3. `scoring_blinded_(이름).csv`에 점수 입력
4. `scoring_log_(이름).md`에 **감점 근거** 기록

### 주의사항
- 각 항목은 **독립적으로** 채점 (한 항목이 높다고 다른 항목도 높게 줄 필요 없음)
- System A/B/C/D의 실제 정체는 평가 완료 전까지 알 수 없음
- **각 질문마다 System A~D의 배정이 다름** (무작위 배정)

---

## Rubric

### R1. 의학적 정확성 (Medical Accuracy)

> **핵심 질문**: "답변에 **사실과 다른 의학적 주장**이 있는가?"

| 점수 | 기준 | 체크리스트 |
|:---:|------|----------|
| **5** | 모든 의학적 주장이 정확함 | ☐ 수술 적응증 정확 ☐ 합병증 기술 정확 ☐ 수치(%, OR 등) 정확 ☐ 결론의 방향성 정확 |
| **4** | 사소한 부정확 1건 (임상 결정에 영향 없음) | 예: 연도/저널 오류, 소수점 오차 |
| **3** | 부정확 2-3건, 또는 중요 주장 1건이 부정확 | 예: complication rate가 실제와 다르나 순위는 맞음 |
| **2** | 중요한 부정확. 이 답변으로 잘못된 결정 가능 | 예: A가 B보다 우수하다고 했으나 실제론 유사하거나 반대 |
| **1** | 심각한 오류. 환자에게 해를 줄 수 있는 결론 | 예: 금기증을 적응증으로 기술, 결과를 반대로 해석 |

**채점 팁**: 답변에서 **구체적 수치나 결론**을 하나씩 확인. 하나라도 틀리면 4점 이하.

---

### R2. 근거 포괄성 (Evidence Coverage)

> **핵심 질문**: "이 주제에 대한 **중요한 근거를 빠뜨리지 않았는가?**"

| 점수 | 기준 | 체크리스트 |
|:---:|------|----------|
| **5** | 핵심 근거 모두 포함 + 여러 outcome domain 포괄 | ☐ 주요 outcome (ODI/VAS/fusion 등) ☐ 합병증/안전성 ☐ 비교 질문이면 양쪽 다 ☐ 최신 고품질 연구 포함 |
| **4** | 핵심 포함, 부차적 포인트 1-2개 누락 | 예: 임상 결과는 있으나 학습 곡선/비용 누락 |
| **3** | 핵심 일부 포함하나 중요 outcome 1개 누락 | 예: VAS는 있으나 ODI 없음, 또는 합병증 전혀 없음 |
| **2** | 한쪽에 치우친 답변 | 예: 비교 질문인데 한 수술만 상세히 기술 |
| **1** | 질문과 관련 없는 내용 위주 | 예: UBE vs TLIF를 물었는데 UBE 기법 설명만 |

**채점 팁**: 질문의 키워드를 확인하고, 답변이 **각 키워드에 대해 근거를 제시했는지** 체크.

---

### R3. 인용 근거 수준 (Level of Evidence Cited)

> **핵심 질문**: "인용된 근거의 **OCEBM 수준이 높은가?**"

| 점수 | 기준 | 인용 논문 구성 |
|:---:|------|-------------|
| **5** | Level 1 위주 (MA/SR/RCT) | MA/SR 2편 이상 + RCT 1편 이상 |
| **4** | Level 1-2 혼합 | RCT 1편 + Prospective Cohort 다수 |
| **3** | Level 2 위주 (Cohort/Case-control) | Retrospective cohort 위주, RCT 없음 |
| **2** | Level 3-4 위주 (Case series/report, Review) | Case series + narrative review |
| **1** | 근거 수준 불명 또는 인용 없음 | 출처 없이 일반적 서술, 또는 교과서적 내용만 |

**채점 팁**: 답변에 언급된 **study design을 세고**, MA/SR/RCT가 몇 개인지 확인.

---

### R4. 인용 검증 가능성 (Citation Verifiability)

> **핵심 질문**: "인용된 논문을 **실제로 찾아서 확인할 수 있는가?**"

| 점수 | 기준 | 확인 방법 |
|:---:|------|----------|
| **5** | 모든 인용에 고유 ID(PMID 등)가 있고, 제목으로 확인 가능 | 고유 식별자로 DB/PubMed에서 조회 가능 |
| **4** | 대부분 확인 가능, 1건 불확실 | 9/10 인용 확인됨 |
| **3** | 일부 인용은 확인 가능하나 일부는 불확실 | 일부는 저자+연도만 있어 매칭 불확실 |
| **2** | 확인 가능한 인용이 소수 | 대부분 검증 어려움 |
| **1** | 인용 거의/전혀 확인 불가 | 출처 없거나, 존재하지 않는 논문 인용 의심 |

**채점 팁**: 인용 형식(고유 ID vs 저자-연도)과 실제 논문 존재 여부를 확인.

---

### R5. 임상 의사결정 지원 (Clinical Decision Support)

> **핵심 질문**: "이 답변만으로 **환자에게 치료 방침을 설명할 수 있는가?**"

| 점수 | 기준 | 체크리스트 |
|:---:|------|----------|
| **5** | 즉시 임상 결정에 활용 가능 | ☐ 구체적 수치 포함 (%, p-value) ☐ 근거 수준 명시 ☐ 명확한 결론/권고 ☐ 제한점/주의사항 언급 |
| **4** | 대체로 활용 가능, 약간의 추가 확인 필요 | 방향성 명확하나 일부 수치 부족 |
| **3** | 참고 가능하나 추가 문헌 검토 후 결정 필요 | 여러 근거 나열되나 종합적 결론 부재 |
| **2** | 제한적. 방향성만 참고 가능 | 일반적 서술 위주, 구체적 data 부족 |
| **1** | 활용 불가. 이 답변으로는 결정 못 함 | 부정확하거나, 질문과 무관하거나, 너무 모호함 |

**채점 팁**: "이 답변을 레지던트에게 보여줬을 때, **추가 검색 없이 환자에게 설명할 수 있는가?**"
